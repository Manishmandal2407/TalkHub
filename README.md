TalkHub – Real-time Messaging System 	
•	Built a real-time messaging system using Socket.IO enabling duplex WebSocket communication.
•	Added private and group chat features using Socket.IO Rooms (socket.to(), socket.join()).
•	Secured WebSocket connections using JWT authentication in Socket.IO middleware.
•	Handled connection events and configured CORS options to ensure stable and secure communication.

Tech Stack
Node js
express Js
Socket.io
Jwt Authentication
WebSockets
